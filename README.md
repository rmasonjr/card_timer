## HomeSeer HS4 Timer

This is a dashboard card for use with displaying a HomeSeer HS4 Timer on a dashboard.
**Note**::  A regular HS4 Timer device is updated every 1-2 seconds.  It is not practical to update a dashboard this often, 
especially if you have multiple timer cards on a single dashboard; it will bog your system down.
For this reason, the card can be updated every :15 minutes (default) or any other interval down to :1 min only.

**Features**: Animation when timer is updated (user configurable). Uses font-awesome so user can set their own icons.

**Prerequisites**: 
When you create your HS4 timer, make sure you create a corresponding HS4 device.  You can verify this in 
Setup -> General -> "Create Devices for Counters and Timers"

**Installation**: Download the card_timer.zip file attachment from my github.
Go into Edit mode on your dashboard, and click the '+' button on the bottom right.
Click the Upload Card button on the top right and upload the card_timer.zip file you just downloaded.
Add the Timer to your dashboard and configure. Select the Timer and configure as needed
